# Contributing Guidelines

Thank you for visiting **AutoWeld-Vision**. 

This repository is a personal academic research project developed to demonstrate unsupervised anomaly detection pipelines and industrial compliance integration for my master's application and automotive manufacturing recruitment in Germany. 

As a result, I am not actively seeking major pull requests or looking to build a large open-source community around this repository. However, I highly value professional feedback, technical discussions, and collaborative learning.

### How to Engage
- **Report Bugs & Issues**: If you discover an issue while setting up the virtual environment, running the unit tests, or compiling ONNX/TensorRT graphs, please open a detailed issue. Include your system configuration, Python version, and execution logs.
- **Provide Technical Feedback**: If you are a computer vision researcher, quality control specialist, or welding supervisor, I would love to hear your thoughts on coreset memory-bank reduction, late-fusion weighting, or IATF 16949 audit logs. Feel free to open a Discussion topic or reach out via email.
- **Academic Reuse**: If you are working on a similar anomaly detection project, feel free to fork this repository and adapt the late-fusion pipeline under the standard MIT License.
