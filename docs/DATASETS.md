# Dataset Download Guide

## MVTec AD (Required for baseline)
- Official: https://www.mvtec.com/company/research/datasets/mvtec-ad
- Size: ~300MB
- Categories: 15 (bottle, cable, capsule, etc.)
- License: Free for academic use

## MVTec AD 2 (For lighting robustness)
- Official: https://www.mvtec.com/company/research/datasets/mvtec-ad-2
- Size: ~2GB
- Note: Requires submission to evaluation server for private test scores

## VisA (For cross-dataset validation)
- GitHub: https://github.com/amazon-science/visa-anomaly-detection
- Size: ~500MB

## NEU Surface Defect (Weld-relevant)
- Link: http://faculty.neu.edu.cn/songkechen/zh_CN/zdylm/65fb5f6a6
- Size: ~50MB

## GDXray (Weld radiographs)
- Official: https://github.com/computervision-xray-testing
- Size: ~1GB
